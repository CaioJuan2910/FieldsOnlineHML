Seção 02

🖥️ Verificação do Servidor

Análise de impacto no lado servidor — confirmação de que nenhuma alteração é necessária.

O HUD lê dados de $game_actors[1] — estrutura client-side nativa do RPG Maker VX Ace. O servidor envia HP, MP, EXP e Level via pacotes TCP que atualizam automaticamente essa estrutura no cliente. O Sprite_HUD apenas lê esses valores para renderização visual — nenhuma escrita ou comunicação direta com o servidor é realizada. Nenhuma alteração no servidor é necessária.

Arquivo Servidor Relevância Impacto
Server/Scripts/Client/game_character.rb Envia stats do personagem 🟢 Nenhum — apenas leitura
Server/Scripts/Client/game_client.rb Gerencia cliente conectado 🟢 Nenhum — apenas leitura
Server/Scripts/Combat/* Processa batalha/dano 🟢 Nenhum — resultado já em $game_actors[1]

Seção 06

🧩 Estrutura do Módulo HUD_Config

O módulo centraliza todas as constantes de configuração do HUD, permitindo ajustes rápidos sem tocar na lógica do script.

HUD_Config — Estrutura do Módulo · [VS] Sprite_HUD.rb Ruby
module HUD_Config # ── Posição e dimensões ────────────────────────────────────── HUD_X, HUD_Y, HUD_Z, HUD_WIDTH, HUD_HEIGHT # ── Retrato ────────────────────────────────────────────────── FACE_X, FACE_Y, FACE_SIZE, SHOW_FACE # ── Barras HP/MP ───────────────────────────────────────────── BAR_X, BAR_HP_Y, BAR_MP_Y, BAR_WIDTH, BAR_HEIGHT # ── Barra EXP ──────────────────────────────────────────────── EXP_WIDTH, EXP_HEIGHT, EXP_FIXED_X, EXP_FIXED_Y # ── Nível ──────────────────────────────────────────────────── LEVEL_X, LEVEL_Y, LEVEL_W # ── Fonte ──────────────────────────────────────────────────── FONT_NAME, FONT_SIZE, FONT_BOLD # ── Visibilidade ───────────────────────────────────────────── SHOW_HP, SHOW_MP, SHOW_EXP, SHOW_LEVEL # ── Fundo ──────────────────────────────────────────────────── USE_BACKGROUND # ── Movimento (v2.2.0) ─────────────────────────────────────── DRAGGABLE = false # ← HUD e EXP bar fixas # ── 14 constantes de cor (COLOR_*) ─────────────────────────── # COLOR_HP_FILL, COLOR_HP_BORDER, COLOR_HP_BG # COLOR_MP_FILL, COLOR_MP_BORDER, COLOR_MP_BG # COLOR_EXP_FILL, COLOR_EXP_BORDER, COLOR_EXP_BG # COLOR_HP_CRITICAL, COLOR_LEVEL_TEXT # COLOR_SHADOW, COLOR_BG_FILL, COLOR_BG_BORDER end
