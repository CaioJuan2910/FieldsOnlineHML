Etapas de Instalação

1
Adicionar o arquivo [FO] Sprite_HUD.rb ao projeto 🔴 CRÍTICO

Copiar o arquivo do script para a pasta correta do projeto e garantir nomenclatura adequada.

  • Copiar o arquivo [FO] Sprite_HUD.rb para a pasta Client/Scripts/
  • Nomenclatura correta: prefixo [FO], classe FO_Sprite_HUD, módulo FO::
  • O script usa alias sobre Scene_Map — deve ser carregado DEPOIS de [VS] Scene_Map
ℹ️ O prefixo [FO] identifica scripts customizados do projeto Fields Online, diferenciando-os dos scripts base [VS] do VXA-OS.
2
Registrar no !script_order.csv 🔴 CRÍTICO

Este é o problema crítico identificado: script ausente no CSV de ordem de carregamento. Sem este registro, o script não será carregado pelo VXA-OS.

  • Abrir o arquivo Client/Scripts/!script_order.csv
  • Localizar a seção Scenes~blank
  • Inserir [FO] Sprite_HUD APÓS [VS] Scene_Map e ANTES da seção Main~blank
📄 !script_order.csv — Diagrama de Ordem
▼ Scenes~blank
· [VS] Scene_Base
· [VS] Scene_Login
· [VS] Scene_Character
· [VS] Scene_Map
→ [FO] Sprite_HUD ← INSERIR AQUI
▼ Main~blank
· no_script_name
· [RM] Main
⛔
AVISO: Inverter a ordem causará NameError em tempo de execução — o alias tentará referenciar Scene_Map antes de sua definição.
3
Aplicar Fix: método change_opacity 🟡 AJUSTE

Problema identificado: [VS] Scene_Map chama change_opacity() sem argumento em determinados contextos, causando ArgumentError. A correção adiciona um valor padrão ao parâmetro.

✗ ANTES
def change_opacity(value) # sem valor padrão @hud_sprite.opacity = value end
✓ DEPOIS
def change_opacity(value = 255) # valor padrão = opacidade total @hud_sprite.opacity = value end
💡 Sem argumento: mantém opacidade total (255) — HUD sempre visível. Compatível com todas as chamadas existentes no código base.
4
Concluir Etapa 2.3 — Configs Modulares 🟠 PENDENTE

Adicionar ao módulo HUD_Config do script as constantes de configuração modular para tipografia, barras e visibilidade.

  • Adicionar constantes: FONT_NAME, FONT_SIZE, BAR_X, BAR_Y, SHOW_HP, SHOW_MP, SHOW_EXP
Ruby · HUD_Config
module HUD_Config # ── Posição e dimensões ────────────────────────────── HUD_X = 8 # Posição X da HUD na tela HUD_Y = 8 # Posição Y da HUD na tela HUD_Z = 200 # Camada Z (acima de outros sprites) HUD_WIDTH = 240 # Largura total da HUD HUD_HEIGHT = 80 # Altura total da HUD # ── Retrato ────────────────────────────────────────── FACE_X = 6 # Posição X do retrato FACE_Y = 6 # Posição Y do retrato FACE_SIZE = 68 # Tamanho do retrato (quadrado) SHOW_FACE = true # Exibir retrato do personagem # ── Tipografia ─────────────────────────────────────── FONT_NAME = "Nunito" # Nome da fonte utilizada FONT_SIZE = 14 # Tamanho base da fonte # ── Barras ─────────────────────────────────────────── BAR_X = 82 # Posição X das barras BAR_Y = 10 # Posição Y inicial das barras SHOW_HP = true # Exibir barra de HP SHOW_MP = true # Exibir barra de MP SHOW_EXP = true # Exibir barra de EXP # ── Visual ─────────────────────────────────────────── USE_BACKGROUND = true # Exibir fundo semi-transparente DRAGGABLE = false # HUD fixa — jogador não pode mover end
