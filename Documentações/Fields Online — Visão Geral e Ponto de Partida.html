Seção 01

🎮 RPG Maker VX Ace — Visão Geral

Engine de criação de RPGs 2D da Enterbrain/Kadokawa · Lançado em 2011 (JP) / 2012 (Global) · Versão definitiva do VX

⚙️ Principais Características
  • Linguagem: RGSS3 (Ruby Game Scripting System 3), baseado em Ruby 1.9.2 — mais rápido que RGSS2
  • 8 atributos padrão: MHP, MMP, ATK, DEF, MAT, MDF, AGI, LUK
  • Multiple tilesets — retorno ao VX que não tinha
  • 3 camadas de tiles por mapa
  • Character Generator embutido para sprites e faces
  • Sistema de Traits — características especiais de personagens
  • Region IDs para mapas
  • Suporte a vídeo Ogg Theora
  • Gauge de TP além de HP e MP
  • Tiles de 32×32 pixels, resolução customizável via script
🌐 Por que VX Ace para MMORPG?
  • A flexibilidade do RGSS3 permite modificação total do engine via Ruby
  • VXA-OS foi construído exatamente sobre essa base
  • Visual 2D top-down perfeito para MMOs estilo Tibia/Ragnarok
  • Scripts inseridos via Script Editor (F11) em camadas sequenciais
  • Sistema de batalha front-view com battlebacks
  • Evento e Mapeamento melhorado
  • Compatível com VX para arquivos de mapa (mudando extensão)
  • Comunidade ativa com scripts públicos no GitHub
Ruby 1.9.2 RGSS3 2D Top-Down 32×32 Tiles
💡 Nota: Scripts são inseridos via Script Editor (F11) em camadas sequenciais. A ordem dos scripts importa — scripts [RM] primeiro, depois [VS]. Compatível com VX para arquivos de mapa mudando a extensão.
Seção 02

🛡️ VXA-OS — Valentine's Online System

A ferramenta mais completa e segura para criação de jogos online 2D com RPG Maker · MIT License · v2.1.6 (Jan 2022)

Em desenvolvimento ativo desde outubro de 2017, o VXA-OS é construído sobre o RPG Maker VX Ace e oferece uma arquitetura cliente-servidor completa para criação de MMORPGs 2D. Utiliza EventMachine para I/O assíncrono não-bloqueante no servidor e Sequel ORM sobre SQLite (dev) ou PostgreSQL (prod).

🏆 MMORPGs de Referência — Análise e O que Trazer

Os jogos que definem a identidade de Fields Online — mecânicas, sistemas e inspirações diretas

🗡️ Tibia Online
CipSoft · 1997 · Top-Down 2D · Referência Principal
  • Mecânica: Top-down 2D, movimentação por tiles, controle por mouse
  • Vocações: Knight, Paladin, Sorcerer, Druid (cada com promoção)
  • Combate: Turn-based em tiles, magias com mana, ataques físicos
  • Skills: Sword, Axe, Club, Shielding, Fist, Distance, Magic Level
  • PvP: Skull System — branco, amarelo, vermelho, preto
  • Morte: Perda de EXP + possibilidade de drop de itens
  • Minimap: Quadrado, canto superior direito ← referência direta
  • Drops: Timer ~1 minuto, loot visível no chão para todos
O que trazer para Fields Online
💀 Skull System PvP 📉 Morte com perda de EXP 🗺️ Minimap quadrado 🖱️ Movimentação por mouse ⚔️ Skills independentes 😰 Sensação de perigo
⚔️ MU Online
Webzen · 2003 · Isométrico 2D · Hack-and-Slash
  • Mecânica: Isométrico 2D, combate hack-and-slash, drop abundante
  • Classes: Dark Wizard, Dark Knight, Elf, Summoner, Rage Fighter, Magic Gladiator, Dark Lord
  • Progressão: Level + Master Level + Grand Master Level
  • Wings: Asas nível 1, 2, 3 — sinal de prestígio
  • Eventos: Chaos Castle, Blood Castle, Devil Square
  • EXP: Barra em percentual (X.X%) ← já implementado!
  • Itens: +luck, +skill, +excellent options, Chaos Machine
  • PvP: Murder system com penalidade de itens
O que trazer para Fields Online
📊 Barra EXP em % 🌍 Eventos de mundo ✨ Excellent options ⚗️ Chaos Machine 🏰 Devil Square/Blood Castle 🏆 Prestígio visual
🌸 Ragnarok Online
Gravity · 2002 · 2D Sprites / 3D Ambiente · Job System
  • Mecânica: 2D sprites em ambiente 3D, Job System complexo
  • Jobs: Novice → 1st → 2nd → Transcendent → 3rd Job
  • AoE: Storm Gust, Lord of Vermilion, Meteor Storm, knockback
  • Debuffs: Freeze, Stone, Silence, Blind, Poison, Curse, etc.
  • Elemental: Fire, Water, Wind, Earth, Ghost, Undead, Holy, Shadow
  • Craft: Blacksmith e Alchemist como classes de craft
  • Guild Wars: WoE — War of Emperium
  • Social: Party com bônus de EXP, lojas de jogadores
O que trazer para Fields Online
🎭 Sistema de Jobs 💥 AoE em habilidades 🌊 Resistência elemental 🧊 Debuffs ricos ⚔️ War of Emperium 🏪 Lojas de jogadores
💀 Saga Diablo
Blizzard · 1996–2023 · Hack-and-Slash · Loot System
  • Mecânica: Hack-and-slash isométrico, dungeons procedurais
  • Raridade: Normal → Magic (azul) → Rare (amarelo) → Set (verde) → Legendary (dourado)
  • Loot: Drop físico, proteção temporária (D4), bounce animation
  • Morte: Drop do corpo (D2), perda de EXP e durabilidade
  • Craft: Horadric Cube (D2), Kanai's Cube (D3)
  • Dungeons: Instanciadas por grupo, aleatoriedade, dificuldade escalonável
  • Progressão: Skill trees (D2), Paragon/Seasonal (D3/D4)
O que trazer para Fields Online
💎 Raridade de itens 🔒 Proteção de loot 🎯 Bounce animation 🏰 Dungeons instanciadas 💥 Knockback ⚗️ Craft com receitas
🌍 World of Warcraft
Blizzard · 2004 · 3D · Polimento Extremo · Referência de UI
  • Mecânica: 3D, questing, endgame raids, polimento extremo
  • Classes: 13 classes com especialização — árvore de talentos
  • Loot: Bag system, tooltips ricos, Need/Greed roll, Auction House
  • Social: Guilds com banco, conquistas, títulos, montarias
  • Interface: Frames modulares, buffs/debuffs em ícones claros
  • Quest System: Quest log, rastreamento, objetivos no mapa, chains
  • Ciclo: Dia/noite com eventos temporais
  • Minimap: Circular ← Fields usará quadrado (Tibia style)
O que trazer para Fields Online
🌳 Árvore de talentos 🏆 Conquistas 🌙 Ciclo dia/noite 🏪 Auction House 📜 Quest chains 🎲 Need/Greed ⭐ Reputação
