// 01 — VISÃO GERAL

📋 Visão Geral do Projeto

Fields Online é um MMORPG 2D em desenvolvimento ativo, construído sobre a plataforma VXA-OS (Valentine's VXA-OS), que estende o RPG Maker VX Ace com funcionalidades multiplayer online, sistema de rede TCP, banco de dados e arquitetura cliente-servidor completa.

// 06 — ARQUITETURA

🏗️ Arquitetura Cliente-Servidor

Fields Online utiliza uma arquitetura cliente-servidor clássica de MMORPG, onde o cliente RPG Maker VX Ace se comunica com o servidor Ruby via protocolo TCP/IP, trocando pacotes de dados em tempo real para sincronização do estado do jogo.

⚔️
📡
Camada de Rede
Comunicação TCP/IP bidirecional em tempo real. O cliente envia ações do jogador e recebe atualizações do estado do mundo de todos os jogadores conectados.
🗄️
Persistência de Dados
O servidor gerencia banco de dados com contas, personagens, inventário, quests, switches por jogador e estado do mundo persistido entre sessões.
⚔️
Lógica de Jogo
Combate, movimentação de eventos, drops, trades e guildas são processados server-side para garantir integridade e prevenir cheating.
